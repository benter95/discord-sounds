# MP3 Storage for Discord Plugin

This repository is used to store MP3 files that are needed for a Discord plugin I'm working on.

## Usage

The MP3 files stored here will be used in the Discord plugin for various purposes such as sound effects, music playback, or custom alerts.

## How to Add New Files

1. Add your MP3 files to the repository.
2. Make sure the filenames are clear and descriptive.
3. Keep file sizes manageable to avoid performance issues.
